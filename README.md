# SumMag

SumMag: a meta-programming library for Hastega and Cockatorice.

## Installation

This package can be installed by adding `sum_mag` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:sum_mag, "~> 0.0.8"}
  ]
end
```
